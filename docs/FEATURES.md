# Feature Documentation

## Core Features

### 1. Strength Index (SI)
Unified metric representing total strength capacity.

**Calculation:**
- Weighted by muscle groups (legs = 1.2x, arms = 0.7x)
- Normalized by bodyweight
- Based on estimated 1RM from all exercises
- Updates with each workout

**Display:**
- Animated circular progress ring
- Trend indicator (↑/↓/→)
- Percentage change from last week
- Color-coded by progress

### 2. Workout Logging
Track all training sessions.

**Features:**
- Exercise library (40+ exercises)
- Set tracking (reps, weight, RPE)
- Multiple exercises per session
- Auto-save functionality
- Delete/edit capabilities

### 3. Nutrition Tracking
Monitor daily food intake.

**Features:**
- Meal logging with macros
- Visual macro rings
- Daily totals
- Calorie tracking
- Quick meal templates (coming soon)

### 4. Analytics
Visualize progress over time.

**Charts:**
- Strength Index growth (line chart)
- Weekly volume (bar chart)
- Muscle group distribution (pie chart)
- Trend indicators
- Performance metrics

### 5. Dashboard
Central hub for daily overview.

**Widgets:**
- SI ring with live updates
- Quick stats (workouts, nutrition, recovery)
- Today's mission checklist
- Recent activity feed
- AI insights (coming soon)
